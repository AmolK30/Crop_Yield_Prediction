🌾 Crop Yield Prediction using Machine Learning
📌 Overview

This project predicts crop yield using Machine Learning based on factors like rainfall, temperature, crop type, fertilizers, and season. It helps in understanding how different environmental and agricultural factors affect crop production.

📊 Dataset
Source: Kaggle
Contains data on:
Crop
State
Season
Rainfall
Temperature
Fertilizer & Pesticide usage
Area and Yield
⚙️ Technologies Used
Python
Pandas, NumPy
Matplotlib, Seaborn
Scikit-learn
🤖 Machine Learning Models
Linear Regression
Decision Tree Regressor
Random Forest Regressor
📈 Features
Data preprocessing and cleaning
Data visualization (heatmaps, scatter plots, graphs)
Model training and evaluation
Model comparison using MSE and R2 Score
🏆 Results

Random Forest model performed best with higher accuracy and better prediction capability compared to other models.
